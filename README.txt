Cell Length Determination
=========================

This project is an implementation of the CSE 554 final assignment to
determine the length of fruit fly sperm cells from photographs.

Techniques used
---------------

- Adaptive thresholding
- Flooding
- Connected components
- Cell complex
- Thinning
